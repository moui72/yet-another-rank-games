# Feedback

## UX

- [x] F001 In the pairwise ranking view, the "x" on a game in the current
  ranking currently removes it from the list entirely. Instead it should move
  the game to a separate "unranked" collapsible display, not delete it.
  [artifacts: ui]
- [x] F002 Games should initially start in the unranked display and only move
  into the ranked display once they have a rating (i.e. have been part of at
  least one comparison). [artifacts: ui, datamodel]
- [x] F003 Games in the unranked pool need a "trash" button, visually
  distinct from the ranked view's "x", and only the trash button removes a
  game from the list's pool entirely. [artifacts: ui]
