Search BoardGameGeek for a game by name and import it into the catalogue so it can be added to a pool even if it's in no one's collection.
Why: the pool builder can currently only add catalogue games (those already imported by some collection); BGG search closes the gap of adding arbitrary games. Needs the BGG API token.
